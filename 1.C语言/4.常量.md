# 常量

程序运行时，其值不能改变，即为常量。

## 一、常量分类

1. 字面量常量
1. 宏定义常量
1. const常量
1. 枚举常量

## 二、常量定义

### 字面量常量

下面都是字面量常量

```c
100;
666.6';
'a';
"你好";
```

### define常量

#define是预处理指令，会在预处理阶段对源码中的define常量进行替换。

```c
#define Max 100

int main()
{
  printf("%d", Max);
}
```

### const常量

通过const常量定义常量，必须在声明时就赋值，否则之后赋值报错。

```c
int mian()
{
	const int a = 1;
}
```

> define常量和const常量两点区别：
>
> 1. define常量在预编译阶段替换文本，const常量在运行时创建 
> 2. define没有类型限制，const常量可以进行类型检查
